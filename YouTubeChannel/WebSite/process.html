Rule 30: You are NOT ALLOWED to read my process document

The video process is pretty flexible. A bunch of things can happen, but the order is somewhat fluid, at least up to the point of release. Chat GPT is pretty good at answering any questions you have.
  • Come up with ideas (Dragooning the kids into this is very helpful)
  • Write and Refine Script, come up with memes
  • Process source material (sheet music or whatever) into whatever you need
  • Create musx document if appropriate
  • Write parody lyrics if appropriate
  • Get the music in good enough shape for what is needed
  • Film the video
  • Analyze lyrics for the source material (and parody version)
  • Determine changes needed to the website for the video
  • Upload in progress versions of documents to Google Drive as needed
  • Generate shortened links for documents that go in the video description (I use bit.ly)
  • Write video description and any additional commentary
    • Description format: Rule #3, you are NOT ALLOWED to read the video description. [teaser text]
    • Add a link to the transcript / director's commentary if it exists
  • Write transcript from the filmed video
  • Cut video description to 5000 or fewer characters
  • Finalize additional commentary document
    • If there is a transcription put at the top: RULE #13, you are NOT ALLOWED to read my transcripts
    • If there is a director's commentary put at the top: RULE #14, you are NOT ALLOWED to read the director's commentary
  • For sequenced videos, add next/previous links to the previous and instant videos
  • Make public documents in Google Drive available to anyone with the link
  • Publish the video
    • Mark as whether a children's video or not (prevents commenting and playlisting)
    • The video can be published as private as much as needed to correct MISTAKES
    • Thumbnails can be generated using ffmpeg (example command line: ffmpeg.exe -i FileName.mp4 -ss 00:01:01 -vframes 1 Thumb.jpg)
    • Audio can be extracted from video using ffmpeg (example command line: ffmpeg.exe -i input.mp4 -vn -acodec pcm_s16le -ar 44100 -ac 2 output.wav)
  • If the video allows comments, add and pin a comment "Rule #6: You are NOT ALLOWED to comment on my videos!"
  • Make changes to the web site
  • Verify changes are good
  • DISLIKE the video
  • Repeat until heat death of the universe
